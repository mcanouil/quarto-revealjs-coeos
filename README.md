## Installation

This [Quarto](quarto.org) extension can be installed/used using the following command:

- Install
  ```bash
  quarto add mcanouil/quarto-revealjs-coeos
  ```
- Use
  ```bash
  quarto use template mcanouil/quarto-revealjs-coeos
  ```
